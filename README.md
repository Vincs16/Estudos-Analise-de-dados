# Análise de Dados de Vendas de Periféricos

## Contexto
Atue como um analista de dados.  
Sua tarefa é analisar uma base de dados sobre vendas de periféricos de computador para identificar produtos que estão sendo esquecidos pelas pessoas.  

**Base disponível:**  
- ID da venda  
- Loja compradora  
- Produto comprado  
- Quantidade  
- Preço unitário  
- Data da compra  

## Instruções de Análise
- Classificar os feedbacks por produtos menos vendidos.  
- Identificar padrões, problemas, elogios e oportunidades.  
- Apontar evidências nos dados fornecidos.  
- Sugerir ações práticas para os gestores da empresa.
